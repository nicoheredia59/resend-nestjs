## Description

[Resend](https://resend.com) api exmaple built in Nestjs.
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```


## Api Roues
Check '/api' route to view swagger docs

## Env
Create a .env file, there's an .env.example so you can copy and add the resend environment variable.
